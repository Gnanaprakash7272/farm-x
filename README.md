# farm-x

Full farming assistant

Stack: Next.js, Node, PostgreSQL (Prisma)

Quickstart

1. Create a .env file with DATABASE_URL
2. npm install
3. npx prisma migrate deploy
4. npm run dev

API

GET /api/farm - example farm endpoint

Pages added:
- /login
- /permissions
- /dashboard
- /disease (Crop & Vegetable Disease Detection)
